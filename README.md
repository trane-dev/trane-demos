## Local Usage
1. Clone repo
```shell
git clone https://github.com/trane-dev/trane-demos
```
2. Install the requirements
```
pip install -r requirements.txt
```
4. Download the data for the notebooks (put in `/data` folder for each demo)
5. Open Jupyter
```
jupyter notebook
```
